# 茶园植物健康环境监测数据分析

> **选题方向**：自然科学实验与传感器数据分析（方向 10）  
> **数据来源**：Kaggle — Plant Health Data（生物传感器真实采集）  
> **分析工具**：Python + pandas + Matplotlib + Seaborn + Plotly + scipy + scikit-learn

---

## 一、核心分析问题

1. 茶园各环境指标（温度、湿度、土壤湿度、光照）在监测周期内呈现怎样的变化趋势？
2. 不同监测植株之间是否存在显著的环境微差异？
3. 光照强度与叶绿素含量之间是否存在相关性？
4. 不同植物健康状态下，环境与营养指标呈现怎样的分布特征？
5. 能否通过聚类分析自动识别植株环境特征分群？

## 三、数据清洗与处理

### 缺失值
数据集完整性良好，**所有字段均无缺失值**。

### 异常值处理（IQR 法）
对 5 个核心环境指标（土壤湿度、环境温度、土壤温度、空气湿度、光照强度）采用 IQR 方法检测与裁剪。原始数据中异常值极少（多数指标为 0），使用 `clip()` 作边界裁剪而非删除，保证数据完整性。

### 衍生指标
| 指标 | 计算方式 | 含义 |
|------|----------|------|
| Temp_Diff | 环境温度 − 土壤温度 | 气土温差，反映土壤热惯性 |
| Nutrient_Index | (N + P + K) / 3 | 综合营养水平 |
| 健康状态（中文） | Healthy → 健康 / Moderate Stress → 中度胁迫 / High Stress → 高度胁迫 | 图表展示用 |

## 五、核心分析结论

### 结论 1：土壤湿度是影响植物健康状态的最关键因子
ANOVA 分析显示土壤湿度在三组健康状态间差异极显著（**F = 899.26, p < 0.001**）。健康组平均土壤湿度 **34.91%**，中度胁迫 **26.56%**，高度胁迫仅 **18.08%**。湿度每降约 8 个百分点，健康等级下降一级。

### 结论 2：环境温度与土壤温度高度正相关
相关性分析显示气温与土温的 Pearson r **> 0.9**，外部气候对土壤温度影响显著。气土温差可用于监测灌溉需求。

### 结论 3：光照强度对叶绿素含量的线性解释力极弱
回归方程 $y = 0.0007x + 34.35$，**R² = 0.0003**。10 月监测期内光照几乎不影响叶绿素含量，合成受限的主因可能是营养供应或土壤水分。

### 结论 4：茶园内微环境分布不均
各植株间环境指标存在明显差异（图 2、图 5），部分植株持续处于低湿或高温环境，需差异化管理。

### 结论 5：茶园整体健康状态不容乐观
1,200 条记录中健康仅占 **24.9%**（299 条），中度和高度胁迫合计 **75.1%**（901 条），需针对性管理干预。

## 六、项目局限性

1. **时间范围有限**：仅覆盖约一个月（10 月），无法分析季节性变化和年际趋势
2. **样本量有限**：仅 10 株植株，结论外推需谨慎
3. **健康状态标注**：为传感器自动判定，缺乏人工校准，可能存在误分类
4. **指标不全**：缺少 CO₂ 浓度、风速、降雨量等气象因素
5. **因果推断受限**：本分析基于相关性，不能直接推断因果关系
